If a mass conflict event occurs (e.g., a major refactor on master affects many feature branches), the detector could dispatch dozens of resolver workflows simultaneously. GitHub Actions has a limit on concurrent workflow runs per repository (default: 20), and dispatching too many could exhaust the quota, causing legitimate workflow runs to fail.
