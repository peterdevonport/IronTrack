**Option C — Shared mixin via CSS @layer or a helper class:**
Use a dedicated `.theme-light-tokens` class applied to both selectors, keeping token definitions in one place.
```css

  --color-bg: #e2e8f0;
  --color-surface: #ffffff;
  /* ... rest of tokens ... */
}
```
